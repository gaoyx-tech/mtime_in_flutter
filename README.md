# mtime_in_flutter

利用flutter重写之前用kotlin写的电影客户端，接口依然用的是时光网的，目的是练习熟悉使用flutter的各种widget，有些界面使用statefulwidget，有些用的provider，还有一些问题没有解决，之后还会继续修改优化很多不足之处，
纯属个人练习，毫无商业目的，如有侵犯请告知，会立即删除项目。

学习flutter主要通过掘金博客，掘金小册，jspang和blibli的免费视频，都是不错的资源。
